# Зуев Иван Итс-1 лабораторная 3
# задание 1
### текст задачи
Получить список из последних цифр чисел, содержащихся в исходном списке (seq.map)
### Алгоритм решения
1) спрашиваем у пользователя количество элементов в последовательности
2) при помощи цикла просим пользователя ввести данное
кол-во элементов и добавляем их в последовательность
3) используем функцию last для данной последовательности
(функция last оставляет в каждом числе последовательности только последнюю цифру
и выводит пользователю результат)

# задание 2
### текст задачи
Найти сумму тех элементов списка, которые оканчиваются на заданную цифру. (seq.fold)
### алгоритм решения
1) спрашиваем у пользователя количество элементов в последовательности
2) при помощи цикла просим пользователя ввести данное
кол-во элементов и добавляем их в последовательность
3) вызываем функцию sum
(данная функция находит сумму всех элементов последовательности, проверяя условие -
последняя цифра совпадает с цифрой, введенной пользователем( x%10 = m), после чего
выводит результат пользователю )
